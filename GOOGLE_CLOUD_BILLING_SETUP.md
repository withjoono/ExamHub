# Google Cloud 결제 계정 설정 가이드

## 결제 계정 설정 단계

### 1. Google Cloud Console 접속
https://console.cloud.google.com/billing

### 2. 결제 계정 만들기
1. "결제 계정 연결" 또는 "청구 계정 추가" 클릭
2. 국가 선택: 대한민국
3. 결제 정보 입력:
   - 신용카드 또는 체크카드
   - 청구 주소

### 3. 프로젝트에 결제 계정 연결
1. https://console.cloud.google.com/billing/projects
2. "examhub-app" 프로젝트 찾기
3. "작업" > "결제 계정 변경"
4. 방금 만든 결제 계정 선택

### 4. 무료 크레딧 확인
- 신규 사용자: $300 무료 크레딧 (90일간)
- 매월 무료 사용량:
  - Cloud Run: 200만 요청
  - Cloud SQL: 없음 (최소 $10/월)

## 예상 비용

### Cloud SQL (PostgreSQL)
- **db-f1-micro**: ~$10/월 (최소 사양)
- **db-g1-small**: ~$25/월 (권장)
- 스토리지: 10GB 무료, 초과분 $0.17/GB

### Cloud Run
- 무료 티어 내 사용 가능 (200만 요청/월)
- 초과 시: 요청당 $0.40/백만 요청

### 총 예상 비용
- **최소**: 월 $10~15 (무료 크레딧 소진 후)
- **권장**: 월 $25~30
